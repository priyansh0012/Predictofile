\# 🎓 Predictofile: AI-Based Student Dropout Prediction System



🚀 \*\*Live Demo:\*\* https://predictofile.streamlit.app 

💻 \*\*GitHub Repo:\*\* https://github.com/PRIYANSH0012/predictofile  



\---



\## 🚀 Overview

Predictofile is an AI-powered web application that predicts whether a student is at risk of dropping out based on academic and socio-economic factors.



The system not only provides predictions but also explains model behavior and offers actionable insights.
